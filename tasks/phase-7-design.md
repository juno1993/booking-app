## 7.1 공통 레이아웃 & 네비게이션

- [x] 완료

### Header 개선
- 로고 + 사이트명 (좌측)
- **검색바** 중앙 배치 (상품명/지역 검색)
- 카테고리 네비게이션 탭 (펜션 | 호텔 | 공간)
- 우측: 로그인/회원가입 or 유저 드롭다운

### Footer 추가
- 회사 정보, 이용약관, 개인정보처리방침
- 고객센터 연락처
- 소셜 미디어 링크
- Copyright

### 전역 스타일
- Primary 컬러 변경 (야놀자 핑크 or 여기어때 블루 계열 선택)
- 폰트: Pretendard 또는 Noto Sans KR 적용
- 부드러운 그림자 & 라운드 카드 스타일 통일

## 7.2 홈페이지 리디자인

- [x] 완료

### 히어로 섹션
- 풀 너비 배경 이미지/그라디언트
- 캐치프레이즈: "최고의 숙소를 최저가로" 등
- **검색 폼**: 지역/날짜/인원 입력 → 검색 버튼
- 카테고리 퀵 링크 (아이콘 + 텍스트)

### 추천 상품 캐러셀
- 수평 스크롤 or Swiper 캐러셀
- 카드: 이미지 + 상품명 + 가격 + 별점
- "더보기" 버튼

### 카테고리별 섹션
- 펜션 인기 TOP / 호텔 인기 TOP / 공간 인기 TOP
- 각 섹션 3~4개 카드 + 전체보기 링크

### 이벤트 배너
- 프로모션/이벤트 배너 슬라이드

## 7.3 상품 목록 페이지

- [x] 완료

### 필터 & 정렬
- **사이드바 필터**: 카테고리, 가격 범위 슬라이더, 지역
- **정렬 옵션**: 추천순, 가격 낮은순, 가격 높은순, 최신순
- 필터 적용 시 URL 쿼리 파라미터 반영

### 상품 카드 개선
- 이미지 비율 통일 (3:2 or 4:3)
- 이미지 없을 때 카테고리별 기본 placeholder
- 가격 강조 (빨간색 or 볼드 크게)
- 주소 축약 표시 (서울 강남구)
- 하트(찜) 아이콘 (오버레이)
- 카테고리 뱃지

### 페이지네이션
- 무한 스크롤 or 페이지 번호 네비게이션

## 7.4 상품 상세 페이지

- [x] 완료

### 이미지 갤러리
- 메인 이미지 + 썸네일 그리드 (야놀자 스타일)
- 클릭 시 라이트박스 (전체화면 이미지 뷰어)
- 이미지 없을 시 카테고리별 기본 이미지

### 상품 정보 레이아웃
- 좌측 (60%): 이미지 갤러리 + 상세 정보
- 우측 (40%): **예약 카드** (sticky)
  - 가격 표시
  - 날짜 선택 (캘린더 UI)
  - 시간 선택 그리드
  - 예약하기 버튼

### 상세 정보 탭
- **기본 정보**: 설명, 주소 (지도 포함 가능), 운영 시간
- **이용 안내**: 체크인/아웃, 주의사항
- **위치**: 주소 + 카카오맵/네이버맵 연동 (선택)

### 브레드크럼
- 홈 > 펜션 > 상품명

## 7.5 예약 플로우 개선

- [x] 완료

### 날짜 선택 UI
- `input[type=date]` → **캘린더 컴포넌트**로 교체
- 예약 가능 날짜 표시 (초록), 마감 날짜 (회색)
- 오늘 이전 날짜 비활성화

### 시간 슬롯 UI
- 그리드 → 타임라인 뷰 옵션
- 예약 가능: 초록 / 예약됨: 회색 취소선 / 차단: 빨강
- 선택 시 하이라이트 + 가격 표시

### 예약 확인 페이지
- 다이얼로그 → 별도 확인 페이지 or 스텝 형태
- 예약 요약: 상품 이미지 + 이름 + 날짜 + 시간 + 가격
- 예약자 정보 확인 (이름, 전화번호, 이메일)
- 요청사항 입력

### 예약 완료 화면
- 체크 아이콘 + "예약이 완료되었습니다"
- 예약 번호 표시
- 내 예약 보기 버튼

- [x] 완료

### 탭 구분
- 전체 / 예약 대기 / 확정 / 취소됨

### 예약 카드 개선
- 상품 이미지 썸네일 (좌측)
- 예약 정보 (우측): 상품명, 날짜, 시간, 상태 뱃지
- 상태별 색상: 대기(노랑), 확정(초록), 취소(회색)
- 액션 버튼: 취소하기, 상세보기

### 모바일 네비게이션
- 햄버거 메뉴 or 하단 탭 바 (홈, 검색, 내 예약, 마이페이지)

- 하단 고정 예약 버튼 (상세 페이지)

## 7.10 추가 UI 컴포넌트 (필요 시 설치)

```bash
# shadcn/ui 추가 컴포넌트
npx shadcn-ui@latest add carousel       # 캐러셀
npx shadcn-ui@latest add calendar       # 캘린더
npx shadcn-ui@latest add slider         # 가격 슬라이더
npx shadcn-ui@latest add skeleton       # 스켈레톤 로딩
npx shadcn-ui@latest add sheet          # 모바일 사이드 메뉴
npx shadcn-ui@latest add breadcrumb     # 브레드크럼
npx shadcn-ui@latest add pagination     # 페이지네이션
npx shadcn-ui@latest add popover        # 팝오버 (캘린더용)
npx shadcn-ui@latest add command        # 검색 (Cmd+K 스타일)
npx shadcn-ui@latest add scroll-area    # 스크롤 영역

# 외부 라이브러리
npm install swiper                       # 고급 캐러셀/슬라이더
npm install date-fns                     # 날짜 유틸리티
npm install @radix-ui/react-tooltip      # 툴팁
```
